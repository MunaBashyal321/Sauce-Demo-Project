# 🚀 Swag Labs Login Automation

This project automates the login functionality of the **Swag Labs** website using **Selenium WebDriver**, **Java**, **TestNG**, and **Maven**. It demonstrates automated testing of a web application's login feature by validating successful authentication with valid credentials.

---

## 📖 Project Overview

The automation script performs the following actions:

- Launches Google Chrome
- Navigates to the Swag Labs website
- Enters a valid username and password
- Clicks the **Login** button
- Verifies that the user is successfully logged in
- Closes the browser
